
# 🛒 BlinkIT Grocery Sales Analysis Dashboard

## 📌 Overview

This project is a Power BI dashboard created using the BlinkIT Grocery dataset. The dashboard helps analyze sales performance, product categories, outlet performance, and customer ratings to generate meaningful business insights.

## 📂 Files Included

- BlinkIT Grocery Analysis.pbix – Power BI Dashboard
- BlinkIT Grocery Data.xlsx – Dataset used for analysis

## 📊 Key Metrics

- Total Sales
- Average Sales
- Number of Items Sold
- Average Rating

## 📈 Dashboard Insights

- Sales by Item Type
- Sales by Outlet Type
- Sales by Outlet Size
- Sales by Outlet Location
- Sales Trend by Establishment Year
- Customer Rating Analysis

## 🛠️ Tools Used

- Power BI
- Microsoft Excel
- Power Query
- DAX

## 🎯 Objectives

- Analyze overall sales performance
- Identify top-performing product categories
- Compare outlet performance
- Understand customer preferences
- Support business decision-making through data visualization
